Политика конфиденциальности

Дата обновления:

Настоящая Политика конфиденциальности описывает, какие данные обрабатываются в приложении, как они используются и какие права есть у пользователей.

Оператор сервиса: Михаил Шахов
Контактный email: mihaildaikichi@gmail.com

1. Какие данные обрабатываются

  • Имя пользователя, которое вы самостоятельно указываете при первом запуске приложения.
  • Данные, создаваемые вами внутри приложения: задачи (дела), отметки «можно/нельзя», группы, список друзей, история выполнения задач.
  • Технический идентификатор пользователя (UID), создаваемый системой анонимной авторизации.
  • Минимальные технические данные, необходимые для работы сервиса (например, информация об ошибках и сбоях).

2. Авторизация

В приложении используется анонимная авторизация через сервис Firebase, предоставляемый компанией Google.

При анонимной авторизации создаётся технический идентификатор (UID), который не содержит вашего имени, телефона или email и не позволяет напрямую идентифицировать вас как физическое лицо.

3. Какие данные НЕ собираются

  • Мы не запрашиваем номер телефона.
  • Мы не запрашиваем email для входа.
  • Мы не запрашиваем паспортные или платёжные данные.
  • Мы не продаём и не передаём данные третьим лицам в маркетинговых целях.

4. Для чего используются данные

  • Для обеспечения работы приложения.
  • Для отображения созданных вами задач, друзей и групп.
  • Для сохранения истории выполнения задач.
  • Для обеспечения стабильности и безопасности сервиса.

5. Хранение данных

Данные хранятся в инфраструктуре Firebase. Мы стремимся хранить только минимально необходимую информацию для функционирования приложения.

6. Передача данных третьим лицам

Мы не продаём и не передаём персональные данные третьим лицам. Обработка данных может осуществляться поставщиком инфраструктуры (Firebase / Google) исключительно в целях обеспечения работы сервиса.

7. Удаление данных

Вы можете запросить удаление своих данных, написав на mihaildaikichi@gmail.com.

8. Возрастные ограничения

Приложение не предназначено для детей младше 10 лет. Если вы считаете, что ребёнок предоставил данные без согласия законных представителей, пожалуйста, свяжитесь с нами.

9. Изменения политики

Мы можем обновлять настоящую Политику конфиденциальности. Актуальная версия всегда доступна по данной ссылке.

10. Контакты

По всем вопросам, связанным с обработкой данных, вы можете обратиться по адресу: mihaildaikichi@gmail.com.
